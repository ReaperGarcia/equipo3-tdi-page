Acerca de

El propósito de este proyecto práctico es el utilizar herramientas digitales para desarrollar una página web utilizando HTML, con el fin de demostrar la comprensión y aplicación de los conceptos "básicos" de diseño web.

Este proyecto se realizó utilizando el software "Sublime Text", con escritura HTML, CSS y JavaScript.
